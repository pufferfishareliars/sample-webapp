Functions

Different ways of calling functions

The simple way

Simply declare the name of the function, it's parameters, then call it by name.

function functionName(param1, param2, ...){};

functionName(params); // and get the result!

Anonymous Functions

These functions don't have to actually have names. They can anonymously be passed into variables such as below:

var square = function(number){ return number * number };

var x = square(4); // x is assigned value of 16

Function Expressions

Function expressions are very similar to anonymous functions, but can be given a name. This is especially helpful if the function is recursive so it can refer to itself.

var factorial = function fac(n) { return n<2 ? 1 : n*fac(n-1) };

console.log(factorial(3));

You can even reassign it to a new variable and use both later on.

var addtwo = addintTwo(n) { return n + 2};

var addtwoTwo = addtwo;

addtwoTwo(2); // would return 4

Some interesting stuff

Recursion

A function can refer to and call itself. There are three ways for a function to refer to itself:

  1. the function's name
  2. arguments.callee
  3. an in-scope variable that refers to the function

Parameters

Primitive parameters are passed by value, so changes made in the function scope won't reflect in the global scope.

Objects are however passed in by reference, so changes made in the function scope will be reflected.
